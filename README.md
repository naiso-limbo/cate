# 单词猜猜猜

一款通过猜单词的形式提升背单词乐趣的应用
单词猜猜猜是一款由独立开发者精心制作的英语学习软件，区别于传统的背单词应用，我们更关注如何激发用户学习单词的兴趣。
丰富的词库
单词猜猜猜精心筛选当前最常用的英语单词，并且将它们划分为多个等级难度，单词来源均基于当前最新的考纲。用户可以轻松地找到适合自己的词库，并开始学习。
